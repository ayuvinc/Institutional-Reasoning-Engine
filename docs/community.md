# Community and Discussions Setup

This repository supports issues and PR templates out of the box.

## Enable GitHub Discussions (maintainer action)

1. Open repository `Settings`.
2. Scroll to `Features`.
3. Enable `Discussions`.
4. Add categories:
   - Q&A
   - Use Cases
   - Implementations
   - Announcements

## Suggested moderation rules

- Keep posts scoped and evidence-based.
- No legal advice requests; use professional counsel.
- Use issue templates for bugs and document corrections.

## Suggested labels

- `bug`
- `enhancement`
- `question`
- `docs`
- `site`
- `profile-1` ... `profile-6`
